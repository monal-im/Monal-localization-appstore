Quicksy ist ein Ableger des beliebten XMPP-Clients Monal mit automatischer Kontafterkennung.

Du meldest dich mit deiner Telefonnummer an und Quicksy schlägt dir automatisch – basierend auf den Telefonnummern in deinem Adressbuch – mögliche Kontakte vor. Im Hintergrund ist Quicksy ein vollwertiger XMPP-Client, mit dem du mit jedem Benutzer auf jedem öffentlich verbundenen Server kommunizieren kannst. Ebenso können Benutzer auf Quicksy von außen kontaktiert werden, indem du einfach +telefonnummer@quicksy.im zu deiner Kontaktliste hinzufügst.

Abgesehen von der Kontaktsynchronisierung ist die Benutzeroberfläche bewusst so nah wie möglich an Monal gehalten. So können Benutzer schließlich von Quicksy zu Monal wechseln, ohne sich neu in die Funktionsweise der App einarbeiten zu müssen.
