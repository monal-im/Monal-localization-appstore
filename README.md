# Appstore metadata

This repository contains the translations of the appstore metadata of Monal (macOS and iOS) and Quicksy iOS.

## Changing translations already present
Please edit them on Weblate:
- [Monal iOS](https://hosted.weblate.org/projects/monal/appstore-metadata-monal-ios/)
- [Monal macOS](https://hosted.weblate.org/projects/monal/appstore-metadata-monal-macos/)
- [Quicksy iOS](https://hosted.weblate.org/projects/monal/appstore-metadata-quicksy-ios/)

## Adding new translations
Only language codes [listed on this webpage](https://developermemos.com/posts/supported-ios-language-locale-codes-fastlane)
or in the fastlane documentation are supported. We won't merge any PRs adding some language not on that list.
